# MaMoC
A simple money manager terminal application written in C

# How to run
```
$ git clone https://github.com/orfeasmal/mamoc --recurse-submodules && cd mamoc
```

for debug

```
$ make deps
$ ./build/debug/mamoc <optional config file>
```

for release

```
$ make deps release
$ ./build/release/mamoc <optional config file>
```
